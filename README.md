# Studi Kasus

Program pembuatan laporan hasil penilaian mata kuliah dalam satu semester yang menentukan lulus atau tidaknya seorang mahasiswa/i di mata kuliah tersebut.Â Mahasiswa/i dikatakan tidak lulus jika kehadiran kurang dari 12 dan nilai akhir kurang dari 70. Perhitungan nilai akhir yaitu rata-rata dari nilai tugas, nilai UTS dan nilai UAS.

Output dari program berbentuk suatu laporan yang menampilkan nama, total kehadiran, nilai akhir dan predikatnya beserta keterangan lulus atau tidaknya.
